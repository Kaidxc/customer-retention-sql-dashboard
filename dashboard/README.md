# Dashboard Module

## Generated dashboard

`customer_retention_dashboard.html` is generated by the build script and can be opened in a browser.

It contains:

- Executive KPIs: revenue, customers, orders, repeat purchase rate.
- RFM segment revenue distribution.
- Recent monthly KPI table.
- Top campaign target list.
- Cohort retention heatmap.
- Business recommendation.

## Power BI version

For a Power BI version, import these files from `../outputs`:

- `monthly_kpis.csv`
- `rfm_segment_summary.csv`
- `cohort_retention.csv`
- `campaign_targets.csv`
- `customer_metrics.csv`

Recommended pages:

1. Overview - revenue, orders, customers, AOV, repeat purchase rate.
2. Segments - RFM segment size and revenue contribution.
3. Retention - cohort retention heatmap.
4. Campaign Targets - prioritised customer list and targeting logic.

## Suggested visuals

- KPI cards for revenue, customers, orders and repeat purchase rate.
- Bar chart for RFM segment revenue.
- Line chart for monthly revenue and customers.
- Matrix heatmap for cohort retention.
- Table for campaign targets.
